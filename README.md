# FUTURE_PE_02
# UGC Ad Content System — AI Powered
### Built by Mohammed Nishad PA | Future Interns Task 2

A reusable prompt system for generating high-converting 
UGC-style ad scripts using Claude, ChatGPT and Gemini.

## Product Chosen
BrewLocal — D2C Specialty Coffee from Chikkamagaluru, Karnataka

## What is UGC Advertising?
UGC (User Generated Content) ads look like real customer videos,
not traditional ads. They convert better because they feel honest.

## Tools Used
- Claude (claude.ai)
- ChatGPT (chat.openai.com)  
- Google Gemini (gemini.google.com)

## Prompts Built
1. Hook Generator — 5 scroll-stopping hooks
2. Full Script — Problem → Solution → Proof → CTA
3. CTA + Caption Generator

## Framework Used
Problem → Solution → Proof → CTA (PSC Framework)

## Files in This Repo
- /prompts → The 3 master prompts
- /outputs → Generated ad scripts and hooks
